# Установка и требования

EligiusNametag — это современный и мощный плагин. Используя новейшие возможности Minecraft Text API и многопоточность, он требует обновленной среды.

## 📋 Системные требования

| Требование | Минимальная версия | Причина |
|------------|--------------------|---------|
| **Платформа** | Paper 1.21 или Folia 1.21 - 26.1.2+ | Строгое использование компонентов `TEXT_DISPLAY` и команд Brigadier. |
| **Java** | Java 21 LTS | Исходный код скомпилирован в формате JDK 21. |

## 📦 Зависимости

У вас должны быть установлены следующие плагины в папке `plugins/`:

1. **Vault** *(Опционально)*: Если вы хотите назначать определенные дизайны неймтегов в зависимости от ранга пользователя (Admin, VIP и т.д.).
2. **PlaceholderAPI** *(Опционально)*: Для обработки внешних переменных, таких как статистика, деньги или кланы в неймтегах.
3. **ItemsAdder** *(Опционально)*: Для отображения пользовательских эмодзи в именах.

## 🚀 Шаги установки

1. Скачайте файл `EligiusNametag-1.x.x.jar` на вкладке **Releases**.
2. Скопируйте файл в папку `plugins/` вашего сервера.
3. Запустите ваш сервер.
4. Вы увидите анимацию запуска в консоли с котом Eligius MC, подтверждающим успешное подключение плагина.
5. Готово! Вы можете настроить всё в папке `plugins/EligiusNametag/`.
